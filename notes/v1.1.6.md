## v1.1.6 更新日志

本次更新主要带来Windows平台的修复以及一些改进。

### 修复
- 修复Windows平台下初始化可能失败、运行时加载速度慢的问题。

### 改进
- 素材菜单中现在更难误触改变选中状态
- 导出过程中展示等待状态并弹出导出完成的窗口

### 如何使用？
- **Mac（Apple Silicon）** — 下载 `aarch64.dmg` 文件
- **Windows** — 下载 `x64-setup.exe` 文件

## v1.1.6 Changelog

This release primarily brings Windows platform fixes along with some improvements.

### Fixes
- Fixed issues on Windows where initialization could fail and runtime loading was slow.

### Improvements
- The asset menu is now harder to accidentally toggle the selection state
- Export now shows a waiting state and pops up a completion window when done

### How to Use
- **Mac (Apple Silicon)** — Download the `aarch64.dmg` file
- **Windows** — Download the `x64-setup.exe` file
