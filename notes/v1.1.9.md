## v1.1.9 更新日志

本次更新修复了 Windows 平台下 agent 启动时的路径拼接问题。

### 修复
- 修复安装路径包含中文、空格或 Windows 长路径前缀时，Codex、Claude、Gemini、OpenCode 可能无法启动或无法打开登录终端的问题

### 如何使用？
- **Mac（Apple Silicon）** — 下载 `aarch64.dmg` 文件
- **Windows** — 下载 `x64-setup.exe` 文件

## v1.1.9 Changelog

This release fixes Windows path handling when launching agents.
