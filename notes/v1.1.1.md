## v1.1.1 更新日志

本次更新带来多标签页工作流、生成可中止、以及更稳定的跨地域更新分发。

### 新功能
- **多标签页设计** — 支持同时打开多份设计稿，各自独立会话与运行时，互不阻塞
- **可中止的生成** — agent 工作时可一键停止，停止后继续发消息会延续原对话上下文
- **更新源 fallback** — GitHub 之外新增 Cloudflare R2 作为备用下载源，国内网络更稳

### 如何使用？
- **Mac（Apple Silicon）** — 下载 `aarch64.dmg` 文件
- **Mac（Intel）** — 下载 `x64.dmg` 文件
- **Windows** — 下载 `x64-setup.exe` 文件

## v1.1.1 Changelog

This release brings a multi-tab workflow, cancellable generations, and a more resilient cross-region update channel.

### New Features
- **Multi-tab designs** — keep multiple designs open at once, each with its own conversation and runtime, fully concurrent
- **Cancellable generation** — stop the agent mid-run; sending a new message afterwards continues in the same conversation context
- **Update source fallback** — Cloudflare R2 is now a secondary download source alongside GitHub, with seamless failover for blocked networks

### How to Use
- **Mac (Apple Silicon)** — Download the `aarch64.dmg` file
- **Mac (Intel)** — Download the `x64.dmg` file
- **Windows** — Download the `x64-setup.exe` file
