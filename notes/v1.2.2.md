## v1.2.2 更新日志

本次更新修复了 Windows 分发版启动时的内置 Node 调用问题，并改善开发版运行时准备流程。

### 修复
- 修复 Windows 下部分安装路径会导致目录、素材库和设计库加载失败的问题

### 改进
- 开发和构建命令会自动检查并安装缺失的 OpenCode CLI 与 Codex CLI

### 如何使用？
- **Mac（Apple Silicon）** — 下载 `aarch64.dmg` 文件
- **Windows** — 下载 `x64-setup.exe` 文件
