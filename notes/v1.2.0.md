## v1.2.0 更新日志

本次更新带来新的引导式设置流程，并改善 Claude Code 的对话输出体验。

### 新增
- 新增引导式设置流程，帮助首次使用时完成运行时选择、连接检查和首次生成

### 改进
- 改善 Claude Code 工具调用结果、权限拒绝和推理内容在对话中的显示

### 调整
- 移除 Gemini CLI 运行时支持，保留 Codex、Claude Code 和 OpenCode

### 如何使用？
- **Mac（Apple Silicon）** — 下载 `aarch64.dmg` 文件
- **Windows** — 下载 `x64-setup.exe` 文件

## v1.2.0 Changelog

This release adds a guided setup flow and improves the Claude Code conversation experience.

### New
- Added a guided setup flow for runtime selection, connection checks, and the first generation

### Improvements
- Improved how Claude Code tool results, permission denials, and reasoning are shown in the conversation

### Changes
- Removed Gemini CLI runtime support while keeping Codex, Claude Code, and OpenCode

### How to Use
- **Mac (Apple Silicon)** — Download the `aarch64.dmg` file
- **Windows** — Download the `x64-setup.exe` file
