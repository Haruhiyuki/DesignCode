## v1.1.8 更新日志

本次更新集中改善了加载与交互的流畅度。

### 改进
- 切换历史设计稿时保留在历史菜单，并保持原有滚动位置，方便连续浏览历史
- 历史抽屉的侧栏状态在多标签页之间共享，切换标签不再回到“内容”菜单
- 改善了对Claude Code运行时的消息输出的显示

### 修复
- 修复了造成卡顿的一些问题

### 如何使用？
- **Mac（Apple Silicon）** — 下载 `aarch64.dmg` 文件
- **Windows** — 下载 `x64-setup.exe` 文件

## v1.1.8 Changelog

This release focuses on smoother loading and interaction.

### Improvements
- Browsing past designs now stays in the history drawer and keeps its scroll position, so you can continue skimming after opening one
- Sidebar drawer state is shared across tabs — switching tabs no longer bounces you back to the Setup drawer
- Improved how Claude Code runtime output is displayed in the session console

### Fixes
- Fixed several sources of UI stutter

### How to Use
- **Mac (Apple Silicon)** — Download the `aarch64.dmg` file
- **Windows** — Download the `x64-setup.exe` file
